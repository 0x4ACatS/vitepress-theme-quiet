# 🌙 Quiet Tech Blog UI 草图设计文档

> 一个“书桌前的窗户”的博客主题 UI 设计方案

---

# 🪟 一、整体设计概念

## 核心隐喻

用户不是在浏览网页，而是在：

> **夜晚坐在书桌前，通过窗户看向世界，同时阅读自己的思考。**

---

## 关键词

- Minimal
- Calm
- Soft Tech
- Thoughtful
- Anime Background（克制使用）

---

# 🌌 三、顶部区域（窗户 / Hero）

## 设计目标

营造“安静 + 呼吸感”的氛围。

## 视觉元素

- 深色渐变背景：
  - #0F1117 → #161B22
- 微弱星点（低透明度）
- 可选：缓慢移动的云层

## 内容结构

```
[ 标题 ]
Quiet Tech Blog

[ 描述 ]
在混乱的信息世界中...

[ 按钮 ]
→ 进入阅读
```

## 风格说明

- 半透明卡片（类似玻璃）
- 居中布局
- 动效极轻（fade / 呼吸感）

---

# 📚 四、左侧书架（Sidebar）

## 设计隐喻

像一个真实书架，用来整理内容。

## 内容结构

```
📚 分类
  └ 技术
  └ 随笔

🏷 标签
  # VitePress
  # 思考

🧠 Now
  正在写博客
  心情：平静
```

## 交互设计

- hover：轻微高亮（brand color）
- 当前项：轻微突出（像被抽出）

## 视觉建议

- 类似“书脊”的纵向排列
- 间距舒适
- 不使用强分割线

---

# 🪵 五、主内容区（桌面）

## 设计隐喻

桌面上的纸张 / 卡片。

---

## 📄 Post Card 设计

```
┌────────────────────┐
│ 标题               │
│ 一句话摘要...       │
│                    │
│ [标签] [mood]      │
│                    │
│ → 阅读更多         │
└────────────────────┘
```

## 视觉规范

- 背景：#161B22
- 边框：极细 + 模糊
- 圆角：柔和

## 动效

- hover：
  - 轻微上浮（translateY）
  - 边缘发光（brand color）

---

# 📄 六、文章页设计

## 布局结构

```
        标题

   时间 · 标签 · mood

────────────────────

正文内容（680px）

────────────────────

上一篇  |  下一篇
```

## 阅读优化

- 内容宽度：680px – 760px
- 行高：1.6+
- 使用留白而非分割线

---

## ✨ 扩展内容块

### Thinking（思考块）

- 类似便签贴纸
- 可稍微偏色

### Note（笔记块）

- 类似高亮
- 更理性结构

---

# 🌿 七、情绪系统（Mood）

## 示例

```
[今天有点累]
[想明白一点]
```

## 设计建议

- 低饱和颜色
- 小尺寸标签
- 不抢正文注意力

---

# 🎞 八、动效规范

## ✅ 推荐

- hover 微光
- 页面切换 fade
- 卡片轻微浮起

## ❌ 禁止

- 大面积动画
- 滚动特效
- 干扰阅读的视觉效果

---

# 🎨 九、二次元风格使用原则

## ✅ 可以使用

- 窗外背景（天空 / 云）
- 小装饰（书、咖啡、灯）

## ❌ 避免

- 人物立绘
- 高饱和色彩
- 复杂线条

## 核心原则

> 像“动画背景”，而不是“角色插画”。
