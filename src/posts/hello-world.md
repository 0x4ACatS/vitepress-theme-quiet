# 你好，安静

我不确定"开始写博客"这件事需要多少理由。

但我知道，我需要一个地方，可以让我：

- 把模糊的想法变成文字
- 记录技术探索时的思考轨迹
- 对自己坦诚

::: tip 🧠 一个还没想清楚的问题
写给谁看？写给自己吗？写给某个未来的陌生人？还是——写这件事本身就是目的？
:::

也许这个问题不重要。先写就好了。

## 关于这个主题

这个博客主题叫 **Quiet**，因为我希望它是一个安静的地方。

> 在混乱的信息世界中，记录一些有秩序的思考。

深色背景，克制的排版，没有广告，没有推荐算法。只有文字。

::: info Note
如果你也在找一个安静写字的地方，欢迎 fork 这个主题。
:::
